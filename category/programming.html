Markdown syntax cheatsheet

Written by john pfeiffer in programming on Wed 02 July 2014.

  • html <strong> is markdown **strong* or __bold__ = strong or bold
  • html strikethrough is not supported but can just be <del>strikethrough</del> = strikethrough
  • html <blockquote> is markdown > at the start of each line
